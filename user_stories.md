# Smart Clinic Management System - User Stories

### 1. Doctor Dashboard
* **As a Doctor,** I want to log into my portal securely so that I can view my sensitive patient schedules.
* **As a Doctor,** I want to view a list of all my patient appointments for the day so that I can manage my time efficiently.
* **As a Doctor,** I want to write and submit a prescription during an appointment so that the patient can receive their medication.

### 2. Patient Portal
* **As a Patient,** I want to create an account and register my details so that I can access clinic services.
* **As a Patient,** I want to search and find a doctor by their name or specialty so that I can choose the right healthcare provider.
* **As a Patient,** I want to view all the appointments I have booked using my login credentials so that I don't miss my visits.

### 3. Admin Dashboard
* **As an Admin,** I want to add new doctors to the Smart Clinic Management System so that they are available for patient bookings.
* **As an Admin,** I want to manage database configurations and monitor system access to keep clinic data safe.
